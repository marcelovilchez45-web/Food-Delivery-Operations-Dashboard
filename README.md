# 📊 Análisis de Operaciones y Eficiencia de Delivery

## 📝 Descripción del Proyecto
Este proyecto presenta un análisis integral de un dataset con **21,321 registros** de pedidos de comida, abarcando 6 restaurantes únicos (*Swaad, Aura Pizzas, Dilli Burger Adda, etc.*). El objetivo principal fue transformar datos operativos crudos en insights accionables para la optimización logística y financiera.

## 🚀 Desafíos Técnicos y Soluciones
- **Optimización Logística:** Análisis de tiempos de preparación vs. tiempos de espera de repartidores para identificar cuellos de botella.
- **Análisis de Rentabilidad:** Creación de un **Ratio de Empaque** dinámico para medir el impacto del costo operativo por cada dólar vendido.
- **Control de Pérdidas:** Visualización del impacto de cancelaciones y descuentos sobre los ingresos brutos (*Revenue Loss*).

## 🛠️ Tecnologías Utilizadas
- **Power BI:** Modelado de datos y visualización avanzada.
- **DAX:** Implementación de medidas complejas y cálculos dinámicos de promedios con `ALLSELECTED` para análisis de cuadrantes.
- **Limpieza de Datos:** Procesamiento de marcas de tiempo y categorización de distancias.

## 📈 Insights Clave
- Se identificaron los picos de demanda por restaurante y ciudad.
- Correlación directa entre la distancia de entrega y la tasa de satisfacción del cliente.
- Análisis de eficiencia de costos mediante matriz de dispersión (Ventas vs. Ratio de Empaque).
